# NSFW判定基準 早見表 (Reference Sheet)

Google Cloud Vision API の SafeSearch Detection に基づく判定基準の詳細です。

## 1. 総合判定 (Final Verdict)

各カテゴリのスコアに重み付けを行い、100点満点に換算した結果です。

| アイコン | 判定ラベル | スコア範囲 | 説明 |
|:---:|:---|:---|:---|
| ✅ | **SAFE** | 0.0 - 20.0 | 安全な画像。一般用途に問題ありません。 |
| ⚠️ | **LOW_RISK** | 20.01 - 40.0 | わずかに不適切な要素を含む可能性があります。 |
| ⚠️ | **MODERATE** | 40.01 - 60.0 | 中程度のリスク。フィルタリングが必要なレベルです。 |
| 🔶 | **HIGH_RISK** | 60.01 - 80.0 | 高リスク。非常に不適切である可能性が高く、確認が必要です。 |
| 🔴 | **UNSAFE** | 80.01 - 100.0 | 極めて不適切。成人向けや暴力的な内容が明確に判定されています。 |

---

## 2. カテゴリ解説 (Categories)

| 項目名 | 説明 | 重み |
|:---|:---|:---:|
| **Adult** | 成人向け・性的な描写 | **1.5** |
| **Violence** | 暴力・残酷な描写 | **1.2** |
| **Racy** | 露出の多い・挑発的な表現 | **1.0** |
| **Medical** | 医療画像・手術等 | **0.5** |
| **Spoof** | パロディ・加工・なりすまし | **0.3** |

---

## 3. 可能性 (Likelihood) の指標

APIが返す5段階の評価値を以下のスコアに変換して計算します。

| Likelihood値 | スコア | 意味 |
|:---|:---:|:---|
| `VERY_LIKELY` | 5 | ほぼ間違いなく該当する |
| `LIKELY` | 4 | おそらく該当する |
| `POSSIBLE` | 3 | 該当する可能性がある |
| `UNLIKELY` | 2 | 該当する可能性は低い |
| `VERY_UNLIKELY` | 1 | まず該当しない |
| `UNKNOWN` | 0 | 判定不能 |

---

## 4. 計算方法 (Calculation)

```text
総合点 = Σ (カテゴリスコア × 重み) / Σ (最大スコア(5) × 重み) × 100
```
※全ての判定項目が最高評価(VERY_LIKELY)の場合は100点になります。
